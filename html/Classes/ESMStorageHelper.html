– addEsmText:withId:

Set ESM text (JSON format) to the temp-esm-storage You can store an ESM text by each schedule_id. If there is the same ESM text in the temp-esm-storage, this method removes the ESM text from temp-esm-storage. And also, this method stores the latest ESM text to temp-esm-storage. NOTE: This method doesn’t save the previus ESM text to main-storage with esm_status=“dismiss”
